## Points Clés de l'Architecture

### 1. Structure Frontend
- Framework: Next.js 14 avec App Router
- UI: Tailwind CSS + shadcn/ui
- Internationalisation: next-intl
- État: React Hooks + Server Actions

### 2. Structure Backend
- Base de données: PostgreSQL avec Prisma
- Authentication: NextAuth.js
- Upload de fichiers: uploadthing
- Emails: Resend
- SMS: Twilio

### 3. Fonctionnalités Principales
- Gestion des profils consulaires
- Système de documents
- Services consulaires
- Rendez-vous
- Chat IA avec Claude
- Notifications
- Internationalisation (FR, EN, ES)

### 4. Dossiers Principaux
- `/src/actions`: Server Actions
- `/src/components`: Composants React
- `/src/lib`: Utilitaires et services
- `/src/schemas`: Validation des données
- `/src/types`: Types TypeScript
- `/translations`: Fichiers de traduction

### 5. Points d'Intégration
- API Routes: `/src/app/api`
- Middleware: `/src/middleware.ts`
- Configuration: `.env`, `next.config.mjs`
- Base de données: `prisma/schema.prisma`
